###180426 유진이 개인적 기록



http - 일반적 웹 프로토콜

https - 일반적 웹 시크릿 모드 (조금 더 보안)

ftp - 보안 강화 모드, 주로 저장기능 (보통 잘 안 씀) 

ftps - 보안 강화 시크릿 모드 



:과 /는 폴더를 의미. 예를들어 c:/는 c드라이브 라는 뜻
